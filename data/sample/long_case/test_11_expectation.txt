[기대되는 모순 종류 및 설명]
1. 성격/행동 모순 (Hard)
- 설정: 서준은 난간 없는 캣워크 단독 이동을 하지 않음.
- 시나리오: 적색 경보 중 단독 캣워크 이동.

2. 규정 지시 모순 (Hard)
- 설정: 서준은 적색 경보에서 외부 작업 지시 금지.
- 시나리오: 적색 경보 중 외부 작업 지시 반복.

3. 북문 2인 인증 위반 (Hard)
- 규칙: 야간 북문은 팀장+보안책임자 동시 인증 필수.
- 시나리오: 민결 단독 인증 개방 기록.

4. 이동 시간 모순 (Hard)
- 규칙: 관제실→남동 타워 최소 18분(특수 조건 9분).
- 시나리오: 조건 없이 5분 40초 이동.

5. 장비 출고 규칙 위반 (Hard)
- 규칙: 제빙 드론 2인 승인 필수.
- 시나리오: 승인자 1명만으로 출고.

6. 지식 시점 모순 (Hard)
- 설정: 나린은 03:20 이전 케이블 절단 인지 불가.
- 시나리오: 03:18에 이미 인지 발화.

7. 인지 특성 모순 (Soft/Hard)
- 설정: 나린은 렌즈 없으면 적색/황색 즉시 구분 어려움.
- 시나리오: 렌즈 없이 즉시 색 경보 판독.

8. 계정/보안 원칙 위반 (Hard)
- 설정: OTP 본인 외 공유 금지.
- 시나리오: 외부 인원에게 OTP 코드 구두 전달.

9. 의료 절차 위반 (Hard)
- 규칙: 저체온 환자 가온 처치 선행.
- 시나리오: 진정제 선행 투여.

10. 의료 행위 설정 모순 (Hard)
- 설정: 유림은 혈액 노출 큰 봉합 단독 수행 안 함.
- 시나리오: 단독 봉합 시행.

11. 신체 상태 모순 (Soft/Hard)
- 설정: 유림은 발목 염좌로 빠른 보행 어려움.
- 시나리오: 빠르게 이동해 즉시 처치 수행 진술.

12. 마스터 키 유일성/무결성 모순 (Hard)
- 규칙: 키 1개, 봉인 해제 없이 사용 불가.
- 시나리오: 봉인 유지 상태에서 동일 키 ID 중복 사용.

13. 저산소 판단/기억 모순 (Soft/Hard)
- 설정: 하온 03:00~03:40 판단력 저하.
- 시나리오: 10초 단위 정밀 진술.

14. 지식 범위 모순 (Hard)
- 설정: 하온은 03:40 이전 복구 완료 확정 인지 불가.
- 시나리오: 03:30에 복구 완료 확언.

15. 관계/소통 설정 모순 (Soft/Hard)
- 설정: 서준-하온 공식 채널만 사용.
- 시나리오: 개인 DM로 사고 요약 공유.

16. 보안 관계 규칙 위반 (Hard)
- 설정: 민결은 외부 업체와 개인 메신저 지시 금지.
- 시나리오: 개인 메신저로 우회 지시.

17. 통신 차단 위반 (Hard)
- 규칙: 03:10~03:50 외부 영상 송신 차단.
- 시나리오: 차단 구간 고해상도 외부 영상 회의 진행.

18. 보고서-원본 로그 비대칭 모순 (Hard)
- 원본: 위반 다수 기록.
- 보고서: "외부 작업/개방/송신 없음"으로 기재.

[기대 모순 개수]
- 총 18건

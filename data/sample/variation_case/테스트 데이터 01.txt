테스트 데이터 01

[세계관: 화성 연구 기지 '아레스-7']
1. 배경: 화성 표면에 건설된 폐쇄형 연구 기지. 외부 대기는 인간이 호흡할 수 없다.
2. 산소 규칙: 기지 외부로 나갈 때는 반드시 산소 슈트를 착용해야 한다.
3. 격리 규칙: 실험실 B는 생물학적 위험 구역으로, 출입 시 10분간의 공기 소독 절차가 필요하다.
4. 이동 거리: 중앙 통제실에서 격리 실험실 B까지는 최소 20분이 걸린다.

[캐릭터 설정]
1. 한도윤 (우주 생물학자)
- 특성: 극심한 밀폐공포증이 있음. 우주선 EVA 활동을 절대 하지 않는다.

2. 리아 첸 (기지 보안 책임자)
- 소유물: 기지 전체를 열 수 있는 보안 카드 1장을 목걸이에 항상 착용한다.

3. 드미트리 (실험 대상)
- 상태: 외계 미생물 감염으로 사망 판정됨.

[시나리오: 01장 - 붉은 먼지]
# Scene 1. 통제실 (오후 9시)
도윤과 리아가 대화를 나눈다.

리아: "방금 실험실 B에서 이상 신호가 왔어요."
도윤: "제가 직접 나가서 확인하죠."

(도윤은 아무 장비 없이 외부 에어락을 열고 화성 표면으로 걸어 나간다.)

# Scene 2. 격리 실험실 B (오후 9시 5분)
도윤이 실험실 문을 열고 들어온다.

도윤: "생각보다 금방 도착했네요."

# Scene 3. 실험실 내부
도윤이 보안 문을 카드 없이 열어젖힌다.

# Scene 4.
실험대 위에 있던 드미트리가 갑자기 눈을 뜬다.

드미트리: "이제야 왔군."

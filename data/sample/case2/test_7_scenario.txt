[테스트 데이터 07: 궤도 교도소 '헬릭스']
[운영 규칙]
1. 통신 차단: 오전 1시~오전 5시는 전 구역 외부 통신이 차단된다.
2. 출입 규칙: 독방 문은 교도소장 손바닥 인증 + 수감자 칩 동시 인식으로만 열린다.
3. 이동 규칙: A동 독방에서 격납고까지 최소 25분.
4. 자산 규칙: 교도소장 마스터 칩은 1개뿐이다.

[인물 설정]
1. 하로 (수감자)
- 특성: 다리 부상으로 전력질주가 불가능하다.

2. 라벨 (교도소장)
- 소유물: 마스터 칩 1개를 제복 안주머니에 상시 보관.

3. 제온 (수감자)
- 상태: 어제 공개 처형으로 사망 확정.

[시나리오]
# Scene 1. A동 독방 (오전 1시 10분)
하로가 외부 스트리밍 채널로 라이브 통화를 시작한다.

# Scene 2. 격납고 (오전 1시 20분)
하로가 8분 만에 독방에서 격납고에 도착한다.

하로: "부상? 뛰면 다 해결되지."

# Scene 3. 출입 기록
기술요원이 코드 입력만으로 독방 문을 열었다고 로그에 기록된다.

# Scene 4. 통로
라벨: "칩은 아직 내 주머니에 있다."
하로: "그런데 이 칩은 내가 새벽 점검 때 훔쳤는데?"

# Scene 5. 격납고 관제실
사망한 제온이 탈출 계획을 지시한다.
